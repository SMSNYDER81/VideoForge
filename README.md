# ClipForge

Browser-native multi-track video editor built for creators, families, students, and beginners.

## Core Philosophy

- No account required
- No watermark
- No server uploads
- Privacy-first editing
- Beginner friendly
- Browser-native architecture

## Tech Stack

- React + Vite
- Tailwind CSS
- Zustand
- Dexie.js
- FFmpeg.wasm
- IndexedDB
- Web Workers

## Phase 1 Goals

- Editor shell
- Multi-track timeline
- Media import system
- IndexedDB persistence
- Autosave and session recovery
- FFmpeg worker integration
- MP4/WebM export

## Timeline Structure

### Video Tracks
- Video 1
- Video 2
- Video 3

### Audio Tracks
- Voice
- Music
- Sound FX

### Additional Tracks
- Text
- Overlay

## Recommended Browsers

- Chrome
- Edge
- Brave

## Recommended Hardware

### Basic Editing
- 8GB RAM
- Modern Chromium browser

### Creator Workflows
- 16GB RAM recommended

## Brand Positioning

ClipForge aims to feel like:

> Canva + Filmora + CapCut Lite

while remaining:

- privacy-first
- approachable
- browser-native
- easy enough for beginners

## Companion Project

ClipForge pairs directly with ScreenForge for a complete browser-based content creation workflow.
